# Kalkulator BMI

Aplikasi kalkulator BMI sederhana berbasis web untuk menghitung indeks massa tubuh pengguna.

## Fitur
- Menghitung nilai BMI berdasarkan tinggi dan berat badan.
- Memberikan kategori BMI (Kurus, Normal, Gemuk, Obesitas).
- UI sederhana dengan HTML, CSS, dan JavaScript.

## Cara Menjalankan
1. Clone repository ini.
2. Buka file `index.html` di browser. (Gunakan Live Server)
